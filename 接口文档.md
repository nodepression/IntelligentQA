## 前后端接口文档(不断修正)
---
### 全局约定
    1. 如无特殊说明：简单request类型是formdata，参数涉及数组或者其他非简单类型数据的request类型是JSON，response类型是JSON
    2. 返回值形式：
        {
            status:(请求的状态)，
            data:{返回的数据object}
        }
    3. 状态码： 
        200 表示请求成功
        300 表示请求失败
        400 表示没有权限
---
### 登陆注册
    统一前缀 /sign

    1. 登陆
        url : in[post]
        req : 
            username 昵称
            password 密码
     2. 注册
        url : up[post]
        req : 
            username 昵称
            password 密码
            type  用户类型(0:学生 1:教师)
